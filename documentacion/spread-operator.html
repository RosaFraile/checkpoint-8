¿Qué hace el operador de extensión en JS?

1. Introducción

Spread, el nuevo operador de propagación que ofrece JavaScript ES6, nos resultará de ayuda para ahorrar líneas de código a la hora de crear tu proyecto web, ya que genera una lista de valores a partir de un array y es un excelente recurso para mejorar el código JavaScript.

La definición que nos da MDN es: "Spread syntax allows an iterable such as an array expression or string to be expanded in places where zero or more arguments (for function calls) or elements, (for array literals) are expected, or an object expression to be expanded in places where zero or more key-value pairs (for object literals) are expected."

Lo que quiere decir que este operador distribuye los elementos dentro de un iterable (cadena de texto, array o cualquier cosa que se pueda recorrer) dentro de un receptor.

Se puede utilizar para concatenar arrays, crear copias de arrays, convertir cadenas de caracteres en arrays de caracteres unir o clonar objetos, y pasar, de forma dinámica, valores a funciones o constructores, entre otras cosas

El operador Spread simplifica operaciones complejas y proporciona un código más expresivo y eficiente. Esto hace que sea una característica popular entre los desarrolladores de JavaScript.

2. Sintaxis

El operador Spread se denota por tres puntos "..." seguidos de una expresión o un objeto iterable. Es Una herramienta poderosa que proporciona una forma concisa y flexible de trabajar con datos en JavaScript.

La sintaxis del operador Spread es exactamente la misma del parámetro "...rest", aunque en realidad se puede decir que uno hace justo lo opuesto al otro. El operador Spread expande un array en sus elementos, mientras que el parámetro "rest" reune múltiples elementos y los condensa en un solo elemento.

Vamos a ver en que consiste el operador Spread con ejemplos de uso:

3. Copiar un array

  1.  let animales = ['perro', 'caballo', 'gato', 'oso', 'jirafa'];
  2.  console.log(animales); // 'perro', 'caballo', 'gato', 'oso', 'jirafa'
  4.  let copiaDeAnimales = [...animales];
  5.  console.log(copiaDeAnimales); // 'perro', 'caballo', 'gato', 'oso', 'jirafa'

4. Copiar un array y añadirle elementos

Podemos hacer copias de arrays y además añadir elementos al array, tanto al principio como al final de éste.

  1.  let poblaciones = ['Benicasim', 'Castellón', 'Alcocebre'];
  2.  console.log(poblaciones); // ["Benicasim", "Castellón", "Alcocebre"]
  4.  let nuevasPoblaciones = ['Oropesa', ...poblaciones];
  5.  console.log(nuevasPoblaciones); // ["Oropesa", "Benicasim", "Castellón", "Alcocebre"]
  7.  let masPoblacionesNuevas = [...poblaciones, 'Madrid'];
  8.  console.log(masPoblacionesNuevas); // ["Benicasim", "Castellón", "Alcocebre", "Madrid"]

5. Concatenar arrays

Puedes utilizar el operador Spread para incluir los elementos de un array en otro array. Esto es particularmente útil para concatenar arrays o hacer copias de arrays que no tengan otros arrays anidados.

  1.  let peces = ['emperador', 'salmón'];
  2.  let insectos = ['hormiga', 'mosquito'];
  3.  console.log([...peces, ...insectos]); // ["emperador", "salmón", "hormiga", "mosquito"]

Esto también funciona con arrays de diferentes tipos de datos, por ejemplo:

  1.  let numeros = [1, 2, 3];
  2.  let meses = ['enero', 'febrero', 'marzo'];
  3.  console.log([...numeros, ...meses]); // [1, 2, 3, "enero", "febrero", "marzo"]

6. Convertir cadenas de caracteres en arrays

Puedes utilizar el operador Spread para convertir cadenas de caracteres en arrays de caracteres, que pueden ser manipulados o combinados con otros arrays usando métodos asociados a arrays.

Ejemplo:

  1.  const str = "Hello";
  3.  // Spread characters of a string into an array
  4.  const charArray = [...str];
  5.  console.log(charArray); // ['H', 'e', 'l', 'l', 'o']

Podemos usar el operador Spread también en otros contextos, tales como argumentos de funciones, para pasar valores dinámicamente a una función o constructor.

8. Unir o clonar objetos

Puedes usar el operador Spread para añadir las propiedades de un objeto dentro de otro objeto. Esto es útil para unir o clonar objetos, creando un nuevo objeto con sus propiedades sobreescritas o bien extrayendo de un objeto unas propiedades específicas.

8.1 Ejemplo 1: Unión de objetos

  1.  const obj1 = { a: 1, b: 2 };
  2.  const obj2 = { b: 3, c: 4 };
  4.  // Merge objects using spread operator
  5.  const mergedObj = { ...obj1, ...obj2 };
  6.  console.log(mergedObj); // { a: 1, b: 3, c: 4 }

8.2 Ejemplo 2: Clonado de objetos

  1.  const usuario = {
  2.    nombre: 'Alex',
  3.    apellido: 'Tomás',
  4.    edad: 40
  5.  }
  7.  const copiaDeUsuario = {...usuario};
  8.  console.log(copiaDeUsuario);
  9.  // Resultado
  10.  // Object {
  11.    nombre: "Alex",
  12.    apellido: "Tomás",
  13.    edad: 40
  14.  }

9. Concatenar y añadir nuevas propiedades a un objeto

Tal como se puede hacer con los arrays, pasa con los objetos:

10. Combinando el operador Spread con el parámetro "rest"

El operador Spread puede utilizarse con otras características del JavaScript moderno, tales como deconstrucción de arrays y objetos, permitiendo poderosas técnicas de programación funcional. Te permite extraer y manipular elementos de arrays o propiedades de objetos con una sintaxis concisa y expresiva.

Ejemplo:

  1.  const numbers = [1, 2, 3, 4, 5];
  2.  const [first, second, ...rest] = numbers;
  4.  console.log(first); // 1
  5.  console.log(second); // 2
  6.  console.log(rest); // [3, 4, 5]
